About MSP

This is a one site that provides mappings between packages of similar Java libraries




Main Features

Below are the detailed description of the information provided on this site

Identify the packages

A list of supported Java libraries is provided on this site. Within these supported pairs of libraries, each package has a list of recommended similar packages.

Similarity

For each matching pairs of libraries, the cosine similarity between the packages' information of similar libraries are computed. This can be used as a reference of how similar are the recommended packages.

Search desired packages

Find the similar packages that you need by providing a fully-qualified package name.

Find out the packages that you may need

Find the packages that are commonly used together by providing a fully-qualified package name which are displayed under the packages queried.
